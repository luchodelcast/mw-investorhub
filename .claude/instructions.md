# Claude Code Instructions for mw-investorhub

## Workflow

- Environment: Cloud (Nube) in Claude Code
- Branch: main or feature branches
- Auto-deploy: Netlify

## Steps

1. Clone repo from GitHub
2. Create feature branch if needed
3. Make changes in /src or /docs
4. Test locally
5. Commit with clear messages
6. Push and create PR if team collaboration
7. Merge and deploy

## Important

- Keep code in /src
- Keep docs in /docs
- Update README.md as project evolves
- Use descriptive git commit messages
